Everything You Need for AI Video

From script to finished MP4 — fully automated, fully customizable.

AI Voiceover

Choose from 15+ natural-sounding voices powered by edge-tts. Male, female, and multilingual options with adjustable speed and pitch.

AI Background Images

Every scene gets a unique, contextually relevant background image generated via Pollinations.ai. No stock photos needed.

AI Music

Optional background music integration via AIMLAPI. Choose from ambient, upbeat, cinematic, and more — or bring your own audio.

Smart Script Generation

Choose from 4 template styles: educational, listicle, storytelling, or tutorial. Each generates engaging, well-structured scripts.

Thumbnail Generation

Automatically generates eye-catching, SEO-optimized thumbnails with bold text overlays and vibrant backgrounds.

Multiple Formats

Generate landscape (16:9) for standard YouTube or portrait (9:16) for Shorts, TikTok, and Reels. Same API, different parameter.

Dead Simple Integration

A single endpoint does it all. Pick your language.

POST /v1/video/generate
curl -X POST https://api.toolsthatwork.dev/v1/video/generate \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "topic": "Top 5 Productivity Hacks for Developers", "style": "listicle", "voice": "nova", "format": "landscape", "music": true, "captions": true, "thumbnail": true }'
import requests response = requests.post( "https://api.toolsthatwork.dev/v1/video/generate", headers={ "Authorization": "Bearer YOUR_API_KEY", "Content-Type": "application/json" }, json={ "topic": "Top 5 Productivity Hacks for Developers", "style": "listicle", "voice": "nova", "format": "landscape", "music": True, "captions": True, "thumbnail": True } ) job = response.json() print(f"Job ID: {job['id']}") print(f"Status: {job['status']}")
const response = await fetch( "https://api.toolsthatwork.dev/v1/video/generate", { method: "POST", headers: { "Authorization": "Bearer YOUR_API_KEY", "Content-Type": "application/json" }, body: JSON.stringify({ topic: "Top 5 Productivity Hacks for Developers", style: "listicle", voice: "nova", format: "landscape", music: true, captions: true, thumbnail: true }) } ); const job = await response.json(); console.log(`Job ID: ${job.id}`); console.log(`Status: ${job.status}`);
