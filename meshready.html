Production-Ready Output

Not just a mesh — a complete, game-ready asset package.

4-Level LOD Generation

Every asset ships with LOD0 through LOD3 — from full detail for close-up to optimized meshes for distance rendering. Ready for any LOD system.

PBR Texture Baking

Full PBR texture set: Albedo, Normal, Roughness, Metallic, and Ambient Occlusion maps. 1K, 2K, or 4K resolution options.

Unity / Unreal / Godot

Engine-specific export presets ensure proper material setup, scale, and coordinate system for Unity, Unreal Engine, or Godot.

Clean Topology

AI-generated meshes go through automatic retopology to ensure clean quad-dominant topology suitable for animation and deformation.

Asset Catalog

Browse 500+ pre-generated common game assets — barrels, crates, weapons, trees, rocks — available instantly. No generation wait time.

Multiple Quality Tiers

Choose draft (fast, lower poly), standard (balanced), or high (maximum detail) quality tiers depending on your needs and budget.

Simple API, Powerful Output

One request, one complete asset package.

POST /v1/mesh/generate
curl -X POST https://api.toolsthatwork.dev/v1/mesh/generate \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "prompt": "Medieval wooden barrel with iron bands, weathered", "quality": "standard", "format": "glb", "engine": "unity", "lod_levels": 4, "textures": { "resolution": "2k", "maps": ["albedo", "normal", "roughness", "metallic", "ao"] } }'
import requests response = requests.post( "https://api.toolsthatwork.dev/v1/mesh/generate", headers={ "Authorization": "Bearer YOUR_API_KEY", "Content-Type": "application/json" }, json={ "prompt": "Medieval wooden barrel with iron bands, weathered", "quality": "standard", "format": "glb", "engine": "unity", "lod_levels": 4, "textures": { "resolution": "2k", "maps": ["albedo", "normal", "roughness", "metallic", "ao"] } } ) job = response.json() print(f"Job ID: {job['id']}")
const response = await fetch( "https://api.toolsthatwork.dev/v1/mesh/generate", { method: "POST", headers: { "Authorization": "Bearer YOUR_API_KEY", "Content-Type": "application/json" }, body: JSON.stringify({ prompt: "Medieval wooden barrel with iron bands, weathered", quality: "standard", format: "glb", engine: "unity", lod_levels: 4, textures: { resolution: "2k", maps: ["albedo", "normal", "roughness", "metallic", "ao"] } }) } ); const job = await response.json(); console.log(`Job ID: ${job.id}`);
