API Reference

Welcome to the Tools That Work API documentation. This reference covers all endpoints for ClipCraft, AutoTube, and MeshReady.

Base URL for all requests: https://api.toolsthatwork.dev

Authentication

All API requests require authentication via a Bearer token in the Authorization header.

HTTP Header
Authorization: Bearer YOUR_API_KEY

You can generate API keys from your Dashboard. Each key is scoped to your account and can access all three products based on your plan.

Never share your API key publicly. If you believe a key has been compromised, revoke it immediately from the dashboard and generate a new one.

Rate Limiting

Rate limits depend on your plan tier. Exceeding the rate limit returns a 429 Too Many Requests response.

PlanRate LimitMonthly Quota
Free10 requests/min50 calls
Pro60 requests/min2,000 calls
Enterprise300 requests/minUnlimited

Rate limit headers are included in every response:

Response Headers
X-RateLimit-Limit: 60 X-RateLimit-Remaining: 58 X-RateLimit-Reset: 1714003200

Error Codes

The API uses conventional HTTP status codes. Errors return a JSON body with details.

Error Response Format
{ "error": { "code": "invalid_api_key", "message": "The provided API key is invalid or expired.", "status": 401 } }
CodeMeaningCommon Cause
400Bad RequestInvalid JSON or missing required fields
401UnauthorizedMissing or invalid API key
403ForbiddenPlan does not include this feature
404Not FoundJob or resource does not exist
429Rate LimitedToo many requests — slow down
500Server ErrorInternal error — retry with exponential backoff

ClipCraft API

POST /v1/video/generate

Generate a complete video from a topic. Returns a job ID that can be polled for status.

Request Body

JSON
{ "topic": "Top 5 Productivity Hacks", // required "style": "listicle", // "educational" | "listicle" | "storytelling" | "tutorial" "voice": "nova", // see voice list below "format": "landscape", // "landscape" | "portrait" "music": true, // optional, default false "captions": true, // optional, default true "thumbnail": true // optional, default true }

Response (202 Accepted)

JSON
{ "id": "job_a1b2c3d4e5", "status": "processing", "estimated_time": 120, "poll_url": "/v1/video/status/job_a1b2c3d4e5" }

Try it out

POST
GET /v1/video/status/{job_id}

Check the status of a video generation job. Poll this endpoint until status is completed or failed.

Response (200 OK)

JSON — Completed
{ "id": "job_a1b2c3d4e5", "status": "completed", "video_url": "https://cdn.toolsthatwork.dev/v/abc123.mp4", "thumbnail_url": "https://cdn.toolsthatwork.dev/t/abc123.jpg", "duration": 185, "format": "mp4", "resolution": "1920x1080", "created_at": "2026-04-09T14:30:00Z" }
GET /v1/video/list

Retrieve a paginated list of all generated videos for your account.

Query Parameters

ParamTypeDescription
pageintegerPage number (default: 1)
per_pageintegerResults per page (default: 20, max: 100)
statusstringFilter by status: processing, completed, failed
DELETE /v1/video/{video_id}

Delete a video and its associated assets (thumbnail, audio). This action is irreversible.

Response (204 No Content)

Returns an empty response on success.

AutoTube API

POST /v1/autotube/channels

Create and configure a new automated YouTube channel. Requires OAuth connection for auto-upload.

Request Body

JSON
{ "name": "Space Facts Daily", "niche": "science", "voice": "adam", "style": "cinematic", "formats": ["long", "shorts"], "youtube_oauth_token": "ya29.a0AfH6...", // optional "auto_upload": true }

Response (201 Created)

JSON
{ "id": "ch_x9y8z7", "name": "Space Facts Daily", "niche": "science", "status": "active", "auto_upload": true, "topics_remaining": 50, "created_at": "2026-04-09T14:30:00Z" }
POST /v1/autotube/channels/{channel_id}/schedule

Set or update the publishing schedule for a channel.

Request Body

JSON
{ "frequency": "daily", // "daily" | "every_other_day" | "weekly" "publish_time": "14:00", // UTC time "timezone": "America/New_York", "long_form_ratio": 0.7, // 70% long-form, 30% Shorts "active": true }
GET /v1/autotube/channels/{channel_id}/analytics

Retrieve analytics data for a channel. Includes views, subscribers, watch time, and video performance.

Query Parameters

ParamTypeDescription
periodstring"7d", "30d", "90d", or "all" (default: "30d")

Response (200 OK)

JSON
{ "channel_id": "ch_x9y8z7", "period": "30d", "views": 12480, "subscribers_gained": 341, "watch_hours": 892.5, "videos_published": 28, "top_video": { "title": "10 Facts About Black Holes", "views": 3210 } }

MeshReady API

POST /v1/mesh/generate

Generate a game-ready 3D asset from a text prompt. Returns a job ID for polling.

Request Body

JSON
{ "prompt": "Medieval wooden barrel with iron bands", // required "quality": "standard", // "draft" | "standard" | "high" "format": "glb", // "glb" | "fbx" | "obj" "engine": "unity", // "unity" | "unreal" | "godot" | "generic" "lod_levels": 4, // 1-4, default 4 "textures": { "resolution": "2k", // "1k" | "2k" | "4k" "maps": ["albedo", "normal", "roughness", "metallic", "ao"] } }

Response (202 Accepted)

JSON
{ "id": "mesh_f5e4d3c2", "status": "processing", "estimated_time": 180, "poll_url": "/v1/mesh/status/mesh_f5e4d3c2" }
